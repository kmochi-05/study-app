# study-app

## 学習記録アプリ

### 概要
日々の学習内容（科目・時間・日付）を記録・管理するwebアプリ。
### 作成した目的
日々の学習時間を可視化し、継続的な学習習慣を身に付けるために作成した。

### 使用技術
・Python(Flask)
・SQLite
・HTML / CSS
・Git / GitHub

### 機能一覧
・学習記録の登録
・学習記録の一覧表示
・学習記録の削除
・合計学習時間の表示

### 工夫した点
・作成するwebアプリケーションの内容を考え、軽量で簡単に扱えるSQLiteを使用した。SQLiteを用いてデータの永続化を実現した。
・入力後に一覧画面へリダイレクトすることで操作性を向上させた。
・シンプルで直感的に使えるUIを意識した。

### 実行方法
1.リポジトリをクローン
2.以下のコマンドを実行
python app.py
3.ブラウザで以下にアクセス
http://127.0.0.1:5000
